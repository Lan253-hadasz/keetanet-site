Welcome to the KeetaNet Launch Kit

This repository contains a bilingual static site, token documentation, a manifest, and a minimal smart contract scaffold to get the project started.

  • Static pages: index, system, mythology, whitepaper, signals, tokens, contact (RU/EN)
  • Contracts scaffold: contracts/KEETA.sol and deploy script
  • Docs: docs/MANIFEST.md and README-TOKEN-SUITE.md

How to use

  1. Create or clone the repo.
  2. Edit content in assets/, pages and docs.
  3. Deploy the contract with Hardhat (optional) and update token pages.
  4. Enable GitHub Pages from Settings → Pages → Source: main / root.
